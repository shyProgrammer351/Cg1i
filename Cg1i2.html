알고리즘

 프로그램시작 
    먼저 맵 출력 
    캐릭터를 기존 위치에서 지우고 이동한 새 위치에 출력 
    게임 종료 확인 
    게임 종료 아니면 새 이동 방향 입력 받아서 게임 계속 진행 
 프로그램 종료 
 
 맵 출력 함수 
    첫째 Wall이면 벽 출력 
    둘째 ' '이면 길 출력 
    셋째 Exit이면 문 출력 
 
 맵 이동 함수 
    W는 위로 길 한칸 이동 
    A는 왼쪽으로 길 한칸 이동 
    S는 아래로 길 한칸 이동 
    D는 오른쪽으로 길 한칸 이동 
 
 게임 종료 확인 함수 
    Exit에 도착하면 종료 
    아니면 게임 계속 진행
