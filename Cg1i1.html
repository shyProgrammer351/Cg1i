문제

1.6km까지 기본료가 4800원이고 131m 당 100원이 부과된다고 가정할 때, 이동거리(m)를 입력하면 택시비를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
택시의 이동거리(m)를 입력하세요: 1600
1600m 이동시 택시요금은 4800원입니다 


택시의 이동거리(m)를 입력하세요: 1601
1601m 이동시 택시요금은 4900원입니다 


택시의 이동거리(m)를 입력하세요: 3600
3600m 이동시 택시요금은 6400원입니다

 택시요금 계산하는 함수 
    이동거리가 1.6km 미만이면 1.6km로 초기화 
    택시요금은 1.6km까지 기본료가 4800원이고 131m 당 100원 부과
     
 

프로그램 코드 

	// 파일명 : ./Chapter11/TaxiFare.c

	#include <stdio.h>

	 

	// 택시요금 계산하는 함수 

Fb	int calculateTaxiFare( int distance ) {

		// 이동거리가 1.6km 미만이면 1.6km로 초기화 

		distance = ( distance < 1600 ) ? 1600 : distance;

	 

		// 택시요금은 1.6km까지 기본료가 4800원이고 131m 당 100원 부과 

		int taxiFare = 4800 + ( distance - 1600 + 130 ) / 131 * 100;

		return taxiFare;

Fe	}
